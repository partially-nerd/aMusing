## Setup
### Dependencies
- yarn
- git
- nodejs
### Windows
- Execute: `git clone https://github.com/partially-nerd/aMusing.git`
  `cd aMusing`
  `yarn global add electron` (npm fails to install electron properly. it is a large package)
  `npm i youtube-search-api`
  `npm run-script run`
