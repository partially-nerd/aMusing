## Setup
### Dependencies
- yarn
- git
- nodejs
### Windows
- Execute: `cd Desktop` 
  `git clone https://github.com/partially-nerd/aMusing.git aMusing`
  `cd aMusing`
  `yarn global add electron` (npm fails to install electron properly. it is a large package)
  `npm i youtube-search-api`
  `npm run-script run`
